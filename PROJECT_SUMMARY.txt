                        QUICK START GUIDE
═════════════════════════════════════════════════════════════════════════════

AUTOMATIC START (macOS):
$ chmod +x /path/to/start.sh
$ /path/to/start.sh

MANUAL START:

Terminal 1 - Database:
$ mysql -u root -p
mysql> SOURCE create_waste_management_db.sql;
mysql> SOURCE alter_add_constraints.sql;

Terminal 2 - Backend:
$ cd /path/to/codebase/backend
$ php -S localhost:8000

Terminal 3 - Frontend:
$ cd /path/to/codebase/frontend
$ npm install
$ npm run dev

Then open: http://localhost:3000
