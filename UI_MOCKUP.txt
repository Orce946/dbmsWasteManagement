Green (#10b981):
  - Primary buttons
  - Success badges
  - Active navigation items
  - Positive metrics
  - Primary brand color

Blue (#3b82f6):
  - Secondary buttons
  - Links and navigation
  - Info badges
  - Charts (secondary)
  - Secondary actions

Amber/Yellow (#f59e0b):
  - Warning badges
  - Pending status
  - Caution colors
  - Highlight colors

Red (#ef4444):
  - Danger buttons
  - Delete actions
  - Error states
  - Overdue/critical status

Gray (#6b7280):
  - Disabled states
  - Secondary text
  - Borders
  - Dividers
  - Placeholder text

White (#ffffff):
  - Card backgrounds
  - Modal backgrounds
  - Form backgrounds
  - Button text (on colored)

Dark Gray (#1f2937):
  - Primary text
  - Headings
  - Form labels

Light Gray (#f3f4f6):
  - Page background
  - Table headers
  - Hover states
