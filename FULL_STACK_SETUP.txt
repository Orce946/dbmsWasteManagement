   curl -X POST http://localhost:8000/codebase/backend/citizens \
     -H "Content-Type: application/json" \

   curl -X POST http://localhost:8000/codebase/backend/bills \

   curl -X POST http://localhost:8000/codebase/backend/payments \
     -H "Content-Type: application/json" \
     -d '{"amount":500,"payment_method":"Card","bill_id":1}'

ISSUE: "Port 3000 already in use"

  3. Or use different port: npm run dev -- --port 3001

  2. Or use different port: php -S localhost:8001
