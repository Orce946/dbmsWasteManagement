  1. Check MySQL is running: mysql -u root -p
  2. Check ports are free: lsof -i :8000 :3000
  3. Kill processes if needed: kill -9 <PID>

If no data appears:
  1. Check browser console (F12)
  2. Check Network tab for API errors
  3. Verify MySQL has data: mysql -u root -p wasteManagement

If frontend won't load:
  1. Ensure Node.js is installed: node -v
  2. Reinstall dependencies: npm install
  3. Clear npm cache: npm cache clean --force

Still stuck?
  1. Read FULL_STACK_SETUP.txt Troubleshooting section
  2. Check PHP logs: tail -f /tmp/backend.log
  3. Check browser logs: Open DevTools console
