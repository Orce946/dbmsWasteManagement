Location: Data Model
File: backend/models/Area.php

```php
<?php
class Area {
    private $db;
    
    public function __construct($db) {
        $this->db = $db;
    }
    
    public function getAll() {
        // Build the SQL query
        $query = "SELECT * FROM Area ORDER BY name ASC";
        
        // Execute the query against MySQL
        $result = $this->db->query($query);  // â† Database query!
        
        // Check if query was successful
        if (!$result) {
            return [
                'success' => false,
                'error' => $this->db->error
            ];
        }
        
        // Fetch all rows
        $data = [];
        while ($row = $result->fetch_assoc()) {
            $data[] = $row;
        }
        
        // Return formatted response
        return [
            'success' => true,
            'data' => $data
        ];
    }
}
?>
```

What happens:
  1. getAll() method executes
  2. Builds SQL: SELECT * FROM Area ORDER BY name ASC
  3. Sends query to MySQL database
  4. MySQL returns result set (or empty array)
  5. Loops through each row
  6. Builds response array
